快速上手

  1. 檔案下載 msi安裝檔及測試資料
  2. 安裝 "SimCorpFinder-0.1-amd64.msi"
  3. 雙擊"SimCorp Finder"在桌面上的捷徑
  4. 輸入相應的參數
    參數值
    Target CompanyCoca-Cola Global
    Keywordsdrink tea
    KeyWords(Emphasize)beverage
    KeyWords(Filtered)juice
    Finding Companiesselect "coca_cola.csv"
    Threads Numberdefault is acceptable
    Recrawling Datacheck it
    ## 查閱更多資訊，請閱讀應用程式文件
  5. 點擊"Start Ranking"，並打開輸出文檔

輸出文檔的格式

  • 簡要統計

    • 您可從中確認應用程式篩選參數與您輸入的參數是否吻合

  • 公司排序清單

    • 與您輸入關鍵字越相關的公司將會被排在越前面，每間公司的右側也有相應的相關分數

  • 篩選細節

    • 這部分將顯示出有出現您輸入關鍵字的網頁及關鍵字出現次數，這將使您更方便的重新檢核系統篩選出的結果

使用文件

  • 系統要求

    • 64-bit windows 10

    64-bit windows 7

  • 安裝

    • 您可以透過msi安裝工具進行安裝。安裝完畢之後，SimCorpFinder.exe將被放置在"C:\Program File\SimCorpFinder"資料夾下，也會自動在桌面建立捷徑。另外，一但你啟用應用程式，SimCorpFinderData資料夾將被建立在C曹底下，您也可以在其中找到輸出文檔以及程式執行的紀錄檔。

  • 輸入參數

參數介紹
Target Company
  • 你準備移轉訂價報告的標的公司。
  • 中文是可以被接受的。
Keywords
  • 建議您輸入與標的公司產品高度相關的關鍵字。
  • 請用空白鍵分隔詞彙。
  • 僅接受英文輸入。
KeyWords(Emphasize)
  • 與前一個參數概念相同，但加權兩倍。
  • 僅接受英文輸入。
KeyWords(Filtered)
  • 公司產品資訊中出現這個關鍵字將直接被篩選掉。
  • 請小心謹慎使用這個參數，因為搜索引擎尋找出來的資訊可能非常龐雜，過度使用此參數將可能使您遺漏掉重要的資訊。
  • 僅接受英文輸入。
Output Directory
  • 將儲存應用程式輸出的文檔
  • 建議您使用預設值(C:\SimCorpFinderData\outputs)。
Finding Companies
  • 您透過OneSource篩選出的公司清單。
  • 請注意輸入文檔的格式必須是規定格式的csv檔。您可以透過以下步驟製作這個輸入文檔，首先打開一個空的excel檔案，並使用公司名稱清單填滿此文檔的第一欄，在另存新檔為csv檔。
Threads Number
  • 執行緒數量將與應用程式執行速度相關，建議您調整至與您處理器相同核心的數字。
  • 您也可以透過一步步調高此參數，測試您電腦可接受的最大執行緒數量。如果沒有產生錯誤訊息且成功書出文檔，就代表執行成功。
Recrawling Data
  • 本應用程式將會自動記錄您檢索的資料，如此一來您將不必因為更換關鍵字，而重新蒐集每一間可以公司的資料。
  • 然而，如果應用程式在執行資料蒐集的過程中有出錯，建議您使用這個選項幫助您重新蒐集資料，以增加檢所的準確率。

  • Start Ranking 的訊息視窗

    • 這個訊息視窗會告訴您估計的應用程式執行時間，請您在此期間內耐心等候。一旦程式執行完成，書出文檔所在的資料夾視窗將會自動彈出，您可以在其中找到最近時間的輸出文檔。

  • 更新

    • 一但我們發布新的版本，在打開應用程式時將以訊息視窗通知您。若您希望更新應用程式，建議您先透過下方「解除安裝」步驟解除安裝，但無須刪除C槽下的"SimCorpFinderData"資料夾，然後再至官網下載最新版本的應用程式。

  • 解除安裝

    • 您可透過新增或移除程式移除本應用程式，如果您希望將儲存的資料一併刪除，你可至C槽將"SimCorpFinderData"資料夾刪除。
