Documentation

  • System Required

    • It is tested that this program works on computer with 64-bit windows 10 operating system.

  • Installation

    • It is convenient for you to install the application by msi installer. Using msi installer, the SimCorpFinder.exe will be installed under "C:\Program File\SimCorpFinder", and the shortcut will be built on the desktop automatically. Besides, once you open and run the application the SimCorpFinderData directory will be built under "C:\", which will store the outputfile and program log.

  • Input Parameters

ParameterExplaination
Target Company
  • The company you want to prepare TP report for.
  • Chinese is acceptable.
Keywords
  • The keywords shouuld be highly related to the target company's product.
  • You can only use space to seperate words.
  • English Only.
KeyWords(Emphasize)
  • The same concept with previous prarmeter, but with double weight when retrieving.
  • English Only.
KeyWords(Filtered)
  • The company with such words will be filtered out.
  • Please be very careful to use this function. Because the information crawled from search engine can be quite noisy, if you use this function arbitrarily, you may lost something important.
  • English Only.
Output Directory
  • The directory you want to store the output file.
  • It is recommended to use default value (C:\SimCorpFinderData\outputs).
Finding Companies
  • The companies selected from OneSource.
  • It is important for you to notice the format of this input file. A valid csv file is necessary. To make it, you can open an empty excel file, full the first column with a company name list, and save as a csv file.
Threads Number
  • It refers to the speed of crawling. It is recommended to adjust it to the same number with the number of the core of the cpu on your computer.
  • You can also try to tune it higher step by step to see if it is affordable for your computer to burden. If there is not any error message and the program run successfully, that means it's OK.
Recrawling Data
  • The database will remember what you have searched in a day, which will enable you to reset keywords to retrieve data again without recrawling data.
  • However, if you have trouble crawling data, you may want to recrawl data. Then you can select this option

  • Start Ranking Messagebox

    • The messagebox will calculate the approximate crawling time for you, please be patient to wait until the procees finish. After the process done, the output directory will automatically show up, in which you can find the output file.

  • Update

    • When there are some updates available, there will be a messagebox showing up to notice you. And it is recommended for you to remove the app and reinstall the new edition. You can refer to the Uninstall part below, but you don't have to remove "SimCorpFinderData" directory under "C:\" when updating.

  • Uninstall

    • You can remove this app from "System > Apps & features". If you want complete remove all data create from this app, you can also remove "SimCorpFinderData" directory under "C:\"。
